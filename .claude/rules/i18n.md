# Internazionalizzazione (i18n) - Candidate Review Dashboard

## Lingue Supportate

- **it** - Italiano (default)
- **en** - English
- **ar** - Arabic (RTL)

## File Traduzioni

I file di traduzione si trovano in:
```
/public/locales/
├── it/translation.json
├── en/translation.json
└── ar/translation.json
```

## Utilizzo nei Componenti

```typescript
import { useTranslation } from 'react-i18next';

const MyComponent = () => {
  const { t } = useTranslation();

  return <h1>{t('candidates.title')}</h1>;
};
```

## Aggiungere Nuove Traduzioni

Quando aggiungi testo visibile all'utente:

1. **MAI** hardcodare stringhe in italiano
2. Aggiungi la chiave in **tutti e tre** i file di traduzione
3. Usa chiavi descrittive e gerarchiche

```json
// public/locales/it/translation.json
{
  "candidates": {
    "title": "Candidati",
    "addNote": "Aggiungi Nota",
    "changeStatus": "Cambia Stato",
    "confirmDelete": "Sei sicuro di voler eliminare questo candidato?"
  }
}

// public/locales/en/translation.json
{
  "candidates": {
    "title": "Candidates",
    "addNote": "Add Note",
    "changeStatus": "Change Status",
    "confirmDelete": "Are you sure you want to delete this candidate?"
  }
}

// public/locales/ar/translation.json
{
  "candidates": {
    "title": "المرشحون",
    "addNote": "إضافة ملاحظة",
    "changeStatus": "تغيير الحالة",
    "confirmDelete": "هل أنت متأكد أنك تريد حذف هذا المرشح؟"
  }
}
```

## Cambio Lingua

```typescript
import { useLanguage } from '@/common/language-selector/LanguageContext';

const { language, changeLanguage } = useLanguage();
changeLanguage('en');
```

La preferenza lingua viene salvata in localStorage per utente.
